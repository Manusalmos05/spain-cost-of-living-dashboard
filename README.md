# 📊 España: Evolución del Coste de Vida

## Sobre el proyecto

El aumento del coste de vida es una de las principales preocupaciones económicas en España. Este proyecto analiza la evolución de los precios en diferentes categorías de gasto y transforma datos complejos en información visual clara y accionable mediante un dashboard interactivo desarrollado en Excel.

El objetivo no era únicamente construir gráficos, sino convertir datos en conocimiento, identificar tendencias relevantes y comunicar conclusiones que faciliten la toma de decisiones.

---

## Objetivos

* Analizar la evolución del coste de vida en España.
* Detectar tendencias y patrones de crecimiento de precios.
* Comparar el comportamiento de distintas categorías de gasto.
* Diseñar una visualización clara, intuitiva y orientada a la toma de decisiones.
* Aplicar técnicas de análisis de datos y storytelling visual.

---

## Dashboard

El dashboard permite explorar de forma visual la evolución de indicadores relacionados con el coste de vida, facilitando la identificación de cambios significativos y tendencias económicas.

### Principales funcionalidades

✔️ Indicadores clave (KPIs)

✔️ Comparación entre categorías de gasto

✔️ Visualización de tendencias temporales

✔️ Análisis visual orientado a la toma de decisiones

✔️ Diseño enfocado en la experiencia del usuario

## Principales conclusiones

Entre los hallazgos identificados destacan:

* Incremento sostenido del coste de vida durante el periodo analizado.
* Diferencias significativas entre categorías de gasto.
* Impacto desigual de la inflación según el tipo de consumo.
* Tendencias que permiten anticipar posibles escenarios futuros.
